# docs

Projekttervek és koncepcionális dokumentáció gyűjtőmappája.

## Szerepe

Ide kerülnek azok a szöveges anyagok, amelyek nem futtatható kódok és nem adatállományok, hanem a rendszer tervezését, működését vagy későbbi fejlesztési irányait írják le.

## Almappák

- `plans/`: részletes tervek, működési jegyzetek és kategóriarendszer-vázlatok.
