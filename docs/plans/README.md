# plans

Tervezési dokumentumok és koncepcionális jegyzetek mappája.

## Szerepe

Itt vannak a kategóriarendszerrel, adatletöltéssel, API-val, weboldalakkal és alkalmazásfunkciókkal kapcsolatos jegyzetek.

## Almappák

- `tervek/`: konkrét fejlesztési témákhoz kapcsolódó vázlatok.
- `működés/`: működési modellek, folyamatleírások és vizsgálati jegyzetek.
