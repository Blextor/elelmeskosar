# src

A projekt fő forráskód-mappája.

## Szerepe

Itt találhatók a főbb, élesebb bolti adatgyűjtő és adattisztító szkriptek.

## Almappák

- `markets/`: boltonként rendezett, élesebb adatgyűjtő kódok.

## Kapcsolódó mappák

- `experiments/`: kísérleti, folyamatban lévő vagy egyszeri vizsgálati kódok.
- `tools/`: fejlesztői segédprogramok és kategorizáló próbák.
