# elírt termékek

Kézi javítólisták a SPAR normalizáláshoz.

## Szerepe

Az itt lévő szövegfájlok olyan termékneveket és kivételeket tartalmaznak, ahol a terméknévben szereplő kiszerelés, az API-ból jövő egységadat vagy a valós kiszerelés nem egyértelmű.

## Használat

A `normalize_data_spar.py` innen olvassa a kivétellistákat, és ezek alapján dönt egyes vitás kiszereléseknél.
