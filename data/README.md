# data

Letöltött, szűrt és normalizált adatállományok gyűjtőmappája.

## Szerepe

Ez a mappa nem futtatható kódot, hanem a piaci adatgyűjtések eredményeit tárolja. A feldolgozó szkriptek innen olvasnak vissza korábbi állapotokat, és ide írják az új snapshotokat.

## Almappák

- `markets_data/`: időbélyeges CSV-k a bolti termékadatokról.
