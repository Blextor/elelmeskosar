# Tesco

Tesco API-val és termékadatokkal kapcsolatos kísérletek.

## Szerepe

Ebben a mappában vannak a Tesco kategória- és termékadat-lekérő próbák, a lekért CSV-k, JSON-ok és köztes fájlok.

## Tipikus fájlok

- `termeklek*.py`: termékadat-lekérő próbák.
- `kategoria_konvertalas.py`: kategóriaadatok átalakításával kapcsolatos segédscript.
- `termek_adatok*.csv`: lekért vagy előállított termékadatok.
