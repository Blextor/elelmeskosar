# experiments

Folyamatban lévő és kísérleti munkák gyűjtőmappája.

## Szerepe

Az itt lévő kódok és adatok nem feltétlenül véglegesek. Ide kerülnek a gyors prototípusok, összehasonlító elemzések, térképes próbák és bolti API-kutatások.

## Almappák

- `markets/`: bolti API-khoz kapcsolódó kísérletek.
- `price_map/`: ár- és térképmegjelenítési próbák.
- `same_market_in_different_places/`: azonos boltlánc különböző helyszíneinek összehasonlítása.
- `website/`: webes próbák és prototípusok.
