# price_map

Ár- és térképmegjelenítési prototípusok mappája.

## Szerepe

Itt találhatók a Magyarország- és megyehatár GeoJSON-ok, üzletadatok, valamint több HTML próba az árak térképes megjelenítésére.

## Tipikus fájlok

- `hungary.geojson`, `counties.geojson`: térképi határadatok.
- `uzletek.csv`: üzletlista vagy bolthelyszínek.
- `pm*.html`, `price_map_*.html`: megjelenítési próbák.
