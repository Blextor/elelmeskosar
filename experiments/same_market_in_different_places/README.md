# same_market_in_different_places

Azonos boltlánc különböző helyszíneinek összehasonlítására szolgáló kísérleti mappa.

## Szerepe

Ez a mappa azt vizsgálja, hogy ugyanazon lánc különböző üzleteiben hogyan térnek el a termékadatok, árak, vonalkódok és terméknevek.

## Fontos fájlok

- `smdp.py`: két CSV összehasonlítására szolgáló CLI script.
- `smdp2.py`: továbbfejlesztett vagy alternatív összehasonlító script.
- `szeged*.csv`, `szentendre*.csv`: összehasonlítandó bemeneti adatok.
- `results*/`: futtatási kimenetek.
