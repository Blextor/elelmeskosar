# tools

Segédprogramok és egyszeri fejlesztői eszközök mappája.

## Szerepe

Ez a mappa nem a fő alkalmazáslogika része, hanem olyan kódokat és adatokat tartalmaz, amelyek kategorizáláshoz, teszteléshez vagy korábbi próbákhoz készültek.

## Almappák

- `kategorizalo/`: termékkategorizáló próbák és segédadatok.
