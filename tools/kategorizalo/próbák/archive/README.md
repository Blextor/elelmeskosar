# archive

Kategorizáló próbák archív mappája.

## Szerepe

Régebbi scriptverziók, korábbi JSON kategóriák és egyéb régi segédadatok tárolására szolgál.

## Almappák

- `etc/`: vegyes régi bemenetek és segédfájlok.
- `kategoriak_json/`: korábbi kategória JSON változatok.
- `kat_progs/`: korábbi kategorizáló Python programverziók.
