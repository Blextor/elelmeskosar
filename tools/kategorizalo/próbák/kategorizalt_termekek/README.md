# kategorizalt_termekek

Kategorizált termékcsomagok és kategorizálási tesztkörnyezetek mappája.

## Szerepe

Itt vannak a különböző termékkörökre bontott bemenetek, kimenetek és kategóriadefiníciók.

## Almappák

- `alkoholok/`
- `kave_tea_uditok/`
- `kave_tea_uditok2/`
- `pekaru_sutemeny/`
- `sós_snack/`
- `teszt_kornyezet/`

## Megjegyzés

A ZIP fájlok valószínűleg egy-egy kategorizált termékcsomag archivált változatai.
