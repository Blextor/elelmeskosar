# próbák

Kategorizáló prototípusok és tesztadatok mappája.

## Szerepe

Ebben a mappában találhatók a kategorizálási próbák fő scriptjei, bemeneti CSV-k, kategória JSON-ok és eredményfájlok.

## Fontos fájlok

- `kat25.py`: aktuálisabb kategorizáló próbaverzió.
- `kategoria_szerkeszto.py`: kategóriaszerkesztő segédprogram.
- `kategoriak_*.json`: kategóriadefiníciók különböző verziói.
- `eredmeny*.json`: kategorizálási kimenetek.

## Almappák

- `archive/`: régebbi verziók és korábbi adatállományok.
- `kategorizalt_termekek/`: kategóriánként rendezett tesztcsomagok.
- `kepek/`: letöltött vagy teszteléshez használt termékképek.
- `ktest/`: külön tesztkörnyezetek.
