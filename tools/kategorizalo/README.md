# kategorizalo

Termékkategorizáláshoz kapcsolódó segédprogramok fő mappája.

## Szerepe

Ez a terület a termékkategóriák szerkesztéséhez, teszteléséhez és különböző kategorizáló algoritmusok kipróbálásához készült.

## Almappák

- `próbák/`: konkrét kategorizáló prototípusok, adatok és archív verziók.
