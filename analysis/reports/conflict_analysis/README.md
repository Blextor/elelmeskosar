# konfliktus_elemzes

Termékazonosítási és adatütközési elemzések kimeneti mappája.

## Szerepe

Az itt lévő CSV-k és szöveges összefoglalók azt vizsgálják, hogy két SPAR adatállomány között hol térnek el a terméknevek, vonalkódok vagy képlinkek.

## Tipikus fájlok

- `1_azonos_nev_elter_vonalkod.csv`
- `2_azonos_vonalkod_elter_nev.csv`
- `3_azonos_keplink_elter_nev.csv`
- `output_summary.txt`
