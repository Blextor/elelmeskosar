# reports

Elemzési riportok és generált kimenetek mappája.

## Szerepe

Itt vannak az elemző szkriptek eredményei és az ezekből készült összefoglalók.

## Tartalom

- `termekadat_osszehasonlitas.md`: termékadat-összehasonlítás szöveges összefoglalója.
- `conflict_analysis/`: CSV-k és összefoglalók a termékazonosítási konfliktusokról.
