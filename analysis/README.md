# analysis

Elemző szkriptek és elemzési kimenetek mappája.

## Szerepe

Ez a mappa a termékadatok összehasonlításával, azonosító-stabilitással és adatütközésekkel kapcsolatos munkákat fogja össze.

## Almappák

- `scripts/`: futtatható elemző Python szkriptek.
- `reports/`: elemzési dokumentumok és generált kimenetek.
